# Chat Fusion 🚀

Chat Fusion is a modern, feature-rich Android messaging application designed with a focus on privacy, AI integration, and a premium user experience. Built using Kotlin and powered by Firebase, it pushes the boundaries of traditional messaging apps.

## ✨ Features

### 🤖 AI-Powered Experience
- **Smart Replies:** Context-aware suggestions powered by Google Gemini to keep conversations flowing.
- **Chat Summarization:** Instantly get a summary of long conversations in three bullet points.
- **Tone Analysis:** Understand the emotional subtext of messages before you send them.
- **Emoji Suggestions:** Automatically suggests the perfect emoji based on your message content.

### 🔒 Privacy & Security
- **Burn Mode:** Shake your device to enable "Burn Mode" — messages are deleted after reading or after a short period.
- **Ghost Mode:** Uses front-camera face detection to hide your screen if someone else is looking over your shoulder.
- **Biometric Lock:** Secure individual chat rooms with fingerprint or face authentication.
- **Privacy Overlay:** Automatically obscures chat content when proximity or face sensors detect potential privacy risks.

### 📱 Real-Time Connectivity
- **Live Status:** See if your friends are online, their battery levels, and their connection types.
- **Morse Pings:** Send haptic "Morse Pings" to grab attention instantly.
- **Interactive Profiles:** Rich user profiles with mood indicators and last seen status.

### 🎨 Premium Design
- **Material 3:** A modern, clean interface following the latest Android design standards.
- **Dynamic Theming:** Full support for Light and Dark modes with a vibrant, high-contrast palette.
- **Custom Animations:** Smooth transitions and layout animations for a polished feel.

## 🛠 Tech Stack
- **Language:** Kotlin
- **Architecture:** MVVM (Model-View-ViewModel)
- **Backend:** Firebase (Firestore, Authentication, Storage, Messaging)
- **AI Engine:** Google Gemini (Generative AI SDK)
- **Machine Learning:** ML Kit (Language Identification, Translation, Face Detection)
- **UI Components:** Jetpack (Lifecycle, ViewModel, Navigation, ViewBinding, ConstraintLayout)
- **Image Loading:** Coil

## 🚀 Getting Started

### Prerequisites
- Android Studio Ladybug or newer.
- A Firebase Project (with `google-services.json` placed in the `app/` directory).
- A Gemini API Key (added to `local.properties` or environment variables).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Shoaib2513/ChatFusion-Android.git
   ```
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Set up your `google-services.json` from the Firebase Console.
5. Build and run the app on an emulator or physical device.
